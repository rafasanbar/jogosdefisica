# Quiz interativo sobre as Leis de Newton

Aplicativo web simples com 10 questões sobre as três Leis de Newton para
Dinâmica. Cada pergunta apresenta múltipla escolha e, ao clicar em uma
alternativa, o usuário recebe feedback instantâneo informando se acertou ou errou.

## Como executar

1. Clone ou faça o download deste repositório.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
3. Clique nas alternativas para testar seus conhecimentos e ver as explicações.

## Estrutura do projeto

- `index.html`: marcação principal e template das questões.
- `style.css`: estilos responsivos com suporte a modo claro/escuro.
- `script.js`: lógica para montar o quiz, registrar as respostas e exibir o feedback.

## Conteúdo pedagógico

As dez questões abordam interpretações, exemplos e aplicações quantitativas das três
Leis de Newton, equilibrando conceitos conceituais e situações práticas para fixação
do conteúdo de Dinâmica.
